# Como Configurar o Banco de Dados no MySQL Workbench

Estas são as instruções para se conectar ao banco de dados MySQL que está rodando no contêiner Docker.

### Pré-requisitos
1. Certifique-se de que seus contêineres Docker estão em execução. Você pode iniciá-los com o comando `docker-compose up -d` na raiz do projeto.

### Passos para Configuração

1. **Abra o MySQL Workbench** e clique no sinal de `+` ao lado de "MySQL Connections" para criar uma nova conexão.

2. **Preencha os Parâmetros da Conexão:**

   * **Connection Name:** Dê um nome descritivo, como `Sarah Pilates Docker DB`.
   * **Connection Method:** Mantenha `Standard (TCP/IP)`.
   * **Hostname:** Use `127.0.0.1` ou `localhost`.
   * **Port:** `3306`.
   * **Username:** `user` (este é o usuário que a aplicação usa).
     * *Alternativa: você também pode usar `root` para ter acesso de administrador completo.*
   * **Password:** Clique em **"Store in Vault..."** e digite a senha correspondente:
     * Para o `Username` **`user`**, a senha padrão é **`password`**.
     * Para o `Username` **`root`**, a senha padrão é **`rootpassword`**.
   * **Default Schema:** Você pode deixar em branco ou digitar `sarahpilates` para já se conectar diretamente a este banco de dados.

3. **Teste a Conexão:**
   * Clique no botão **"Test Connection"**. Você deve ver uma mensagem de sucesso.

4. **Salve a Conexão:**
   * Clique em **"OK"** para salvar a nova configuração de conexão.

Agora você pode clicar na conexão recém-criada na tela inicial do Workbench para se conectar ao seu banco de dados, visualizar as tabelas criadas pelo Flyway e executar queries.
